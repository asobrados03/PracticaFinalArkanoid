Aquí se presentan sus controles y como poder cambiar de idioma dentro del juego:
CONTROLES: (para manejar la pelota con la raqueta y reiniciar el nivel)
Para moverte de un lado a otro con la raqueta puedes arrastrar el ratón de un lado a otro o utilizar las teclas de dirección 
derecha e izquierda o las teclas "a" y "d", "a" para moverte a la izquierda y "d" para moverte la derecha, para empezar
a mover la pelota al inicio puedes hacer click izquierdo con el ratón, o darle a la tecla de dirección arriba o a la tecla
"w". Para reinciar el nivel al que se esta jugando pulse "r".
